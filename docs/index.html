Supervised Sparse Auto-Encoders as Unconstrained Feature Models for Semantic Composition

Ouns El Harzli, Hugo Wallner, Yoonsoo Nam, Haixuan Xavier Tao
Paper arXiv Code

Abstract

We introduce decoder-only Supervised Sparse Auto-Encoders (SSAEs) as unconstrained feature models for semantic composition. Unlike standard autoencoders that learn both an encoder and decoder, our approach trains only a decoder alongside a structured sparse latent feature space. Each feature is explicitly tied to human-defined semantic properties — such as hair colour, pose, or action — through a binary mask derived from a compositional category/property structure.

By operating on text embeddings from Stable Diffusion 3.5's T5 encoder, the model learns a sparse feature matrix Y and a linear decoder W that together reconstruct embeddings preserving the semantic content of the original prompt. Because the feature space is interpretable by construction, individual features can be read, edited, or swapped to produce new embeddings — and consequently new images — with targeted semantic changes, all without retraining the model.

Method Overview

The decoder-only SSAE reconstructs an embedding x from a sparse feature vector using:

x̂i = W · σ(Mi ⊙ Yi) + b

where Y is the learned sparse feature matrix (one row per prompt), M is a binary mask derived from the category/property structure, σ is a nonlinear activation (ReLU or LeakyReLU), and W, b are the linear decoder weights and bias.

Model Variants

Trainable Inputs. Learns Y as a full parameter matrix with shared features across prompts via a dedicated y_same vector. The most flexible variant, suited for direct feature manipulation.

Average Feature. Uses an embedding layer to represent per-property features that are composed via the mask. More parameter-efficient for large prompt sets.

Inverse Least-Squares. Learns only Y; the decoder weight W is computed analytically via regularised least-squares (W = (XTX + λI)-1XTY). No gradient descent on W.

Property-Based Editing

Prompts are composed from a structured set of categories, each containing discrete properties. The category/property structure defines the binary mask that governs which features in Y are active for each prompt. By manipulating individual features or swapping property activations, new embeddings can be generated that correspond to targeted semantic edits.
